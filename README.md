## TÍTULO
Caperucita Roja: Una historia alternativa

## DESCRIPCIÓN DEL PROYECTO
Diseño de una página web que narra una historia alternativa de Caperucita Roja. 

## PLANIFICACIÓN
A continuación, mostramos una síntesis de los pasos a seguir. 

Desglose HTML

El trabajo se desarrolla en "dev" vía git branch dev. Cada feature, dispondrá de su propia branch feature. 

1. <head>: encabezado. 
    <title>: texto del encabezado.
    <icon>: icono del encabezado.
Ejemplo:
Inicio desarrollo feature <tittle> en su branch. Tras concluir, git add. Después, git commit.
Inicio desarollo feature <icon> en una nueva branch. Tras concluir, git add. Después, git commit.
Reajuste de ambas features para ajustar <head>. Tras concluir, git add. Después, git commit.
2. <body>
    <header>
        <h1>: Caperucita Roja: Una historia diferente.
    <main>
        <image>: Imagen "El bosque no es lo que parece"
        <h2>: Subtítulo

    Lo descrito en el paso 2 se repetirá con cada una de las escenas siguientes.

    3. <foot>
        <navigation bar>
        <icons>
    
    Lo aquí descrito es susceptible de modificación.
